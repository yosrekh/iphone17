# iPhone 17 Pro Max - Premium Scrollytelling

This is a high-end, 3D animated website for the iPhone 17 Pro Max, built with **Next.js**, **Framer Motion**, and **Canvas** image sequences.

## Features
- **Dynamic 3D Scrollytelling**: 280 high-quality frames transition smoothly as you scroll.
- **Interactive Hotspots**: Hover over features like the Titanium Frame, A19 Pro Chip, and Pro Camera to see details.
- **Glassmorphism Design**: Sleek, Apple-inspired UI with backdrop blurs and premium typography.
- **Ambient Audio**: Background music triggers on first interaction for an immersive experience.
- **Performance Optimized**: Uses WebP image sequences and Next.js Image component for fast loading.

## Getting Started

First, install the dependencies:
```bash
npm install
```

Then, run the development server:
```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Tech Stack
- Framework: [Next.js](https://nextjs.org/)
- Animation: [Framer Motion](https://www.framer.com/motion/)
- Icons: [Lucide React](https://lucide.dev/)
- Audio: [Howler.js](https://howlerjs.com/)
- Styling: [Tailwind CSS](https://tailwindcss.com/)
